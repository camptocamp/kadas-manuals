30 /*
31  * \class QgsNetworkAccessManager
32  * \brief network access manager for QGIS
33  * \ingroup core
34  * \since 1.5
35  *
36  * This class implements the QGIS network access manager. It's a singleton
37  * that can be used across QGIS.
38  *
39  * Plugins can insert proxy factories and thereby redirect requests to
40  * individual proxies.
41  *
42  * If no proxy factories are there or none returns a proxy for an URL a
43  * fallback proxy can be set. There's also a exclude list that defines URLs
44  * that the fallback proxy should not be used for, then no proxy will be used.
45  *
46  */
