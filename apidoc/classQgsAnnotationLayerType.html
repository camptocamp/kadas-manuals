◆ hasLayerProperties()

int QgsAnnotationLayerType::hasLayerProperties ( ) const
inlineoverridevirtual

Resturns whether the layer has a properties dialog: 0 = no, 1 = yes, -1 = unknown Note: the yes/no/unknown return is because this function was introduced after showLayerProperties, which was previously used to handle this by returning false if there are no layer properties.

Returning -1 by default here does not break plugin layers which do not yet support this method.
